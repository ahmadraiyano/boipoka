# Boipoka 🐛📚

Boipoka (meaning "Bookworm" in Bengali) is a React-based web application designed for book lovers. Users can explore a curated collection of books, view detailed information about each, and keep track of their reading journey by adding books to their "Read" list. 

**Live Demo:** [https://serene-pegasus-6faf09.netlify.app/](https://serene-pegasus-6faf09.netlify.app/)

## ✨ Features

* **Book Catalog:** Browse a visually appealing grid of books complete with cover images, authors, tags, and ratings.
* **Detailed Book Views:** Navigate to a dedicated details page for each book to read reviews, check the page count, publisher, year of publishing, and categories.
* **Reading List Management:** * Mark books as "Read" with a single click.
  * Your reading list is saved locally in your browser (via `LocalStorage`), so your data persists even if you refresh the page.
* **Sorting Capabilities:** Sort your listed books dynamically by **Total Pages** or **Ratings**.
* **Responsive Design:** A fully responsive, modern user interface built with Tailwind CSS and DaisyUI.
* **Seamless Routing:** Fast, client-side routing with error handling (404 page) using React Router v7.

## 🛠️ Tech Stack

* **Frontend Framework:** [React 19](https://react.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
* **Routing:** [React Router v7](https://reactrouter.com/)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
* **UI Components:** [React Tabs](https://github.com/reactjs/react-tabs)
